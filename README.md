# File Transfer System (Java - TCP)

This is a simple file transfer project using Java socket programming.

## 📌 Features
- Client sends file to server
- Server receives and saves file
- Uses TCP for reliable communication

## 🛠️ Technologies
- Java
- TCP Sockets

## ▶️ How to Run

1. Compile:
   javac Server.java
   javac Client.java

2. Run Server:
   java Server

3. Run Client:
   java Client

## 📂 Output
- File received as: received_file.txt

## 📚 Concepts Used
- Client-Server Architecture
- Socket Programming
- TCP/IP
